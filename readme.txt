Wordpress图片增强插件，可以将远程图片获取到本地，并且可以对文章的图片打水印。

== Description ==
WppaoImage图片增强插件，来自Wppao团队缘殊开发，只为完成网站运营时必须要的几个图片处理功能，如远程图片本地化，图片加水印等功能。

目前支持的功能：
* 图片文字水印功能
* 图片文字水印字体自选
* 图片文字水印尺寸自选
* 图片文字水印颜色自选
* 图片使用图片水印功能
* 水印位置自选


== Installation ==
1. 在WordPress上传启用WppaoImage网站图片增强插件。
2. 在后台插件菜单激活WppaoImage网站图片增强插件。
3. 选择需要的扩展设置即可使用。
